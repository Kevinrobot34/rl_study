## 強化学習勉強用レポジトリ
主にopen AI gymを利用して強化学習に関連したものを実装し置いておくレポジトリ．

gymの使い方などについては，
cartpoleフォルダのREADMEとcartpoleをランダムに動かすプログラム(`cartpole/cp_random.py`)を
見てください．
その他の手法は基本的にこのプログラムからactionやrewardの決め方を変えただけ．
